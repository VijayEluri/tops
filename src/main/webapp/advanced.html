Invent Your Own Structures!

Instead of submitting a pdb coordinate file for comparision, you can, if you wish, specify the topology using only the abstract 'string' notation described here. Basically, a string is in three parts : a name, a vertex list, and an edge list.

These parts are separated by a single space, and each vertex is from the set V = {N, C, E, e, H, h} - representing the terminii, the strands, and the helices. Strands and helices can be up or down, and the vertex string must start with an 'N' and end with a 'C'. The edge list is an ordered sequence of number pairs separated by a colon (':'), and terminated by a letter from the edge type set E = {'A', 'P', 'R', 'L', 'Z', 'X'} which are Antiparallel hydrogen bond, Parallel hydrogen bond, Right handed chirality, Left handed chirality, and two mixed types : Z and X.

Mixed types (Z and X) are for parallel hydrogen bonds (P edges) that also have a chirality relationship between them. The best example of these are beta-alpha-beta units ('NEhEC 1:3Z' or 'NEhEC 1:3X') which can be left or right handed. Although it would be possible to have antiparallel edges with chiralities in this notation, it doesn't make structural sense since the different chiral types are indistinguishable for antiparallel edges.
