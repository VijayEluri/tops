Bioinformatics Research Centre

TOPS Services at Glasgow University

  • TOPS
  • The main TOPS site is is here at Leeds University, including the structure database.
  • Main Structure Comparison Page (1 x All)
  • Compare protein domain structures against two popular classifications of proteins (CATH and SCOP). CATH is version 2.4 and SCOP is version 1.61.
  • Structure - Structure Comparison (Pair)
  • Upload two structures to be converted to TOPS and compared against each other.
  • 'Classic' Motif Matching Page
  • Match predefined 'classic' motifs (Rosmann folds, TIM barrels, etc) against the classifications, or subsets of them.
  • Pattern discovery for CATH Groups
  • Dynamically generate a common pattern for the representatives of a selected CATH group, and view the diagrams for the common pattern and the protein domains
  • Structure Design
  • Compose a 'string' format topology description and submit it to be matched or compared to structures in the database in order to find structural relatives.
  • Software Page
  • Download the compiled Java class files to run at the command line.
