Essential Software

The 'makefile' for this project is actually an XML file for use with ant or an IDE that is ant-compatible. ant is freely available from the apache free software foundation apache.org

Depending on the package, some parts of the system may require outside libraries. Each chapter describes the dependancies for the package it describes. Notably, the tops.view.tops3D package (see : Chapter 5) needs the Java3D extension, and the necessary jar files with the J3D classes should be in the classpath.

Running ant on the Buildfile

To get a list of possible targets, run ant with just the -projecthelp flag. This should print out the targets and a short description string for each. Note that only targets with a description attribute will be listed. The chapter for a particular package gives the target name needed to build that package at the end of the chapter.

Once ant is properly set up (with the environment variables ANT_HOME and JAVA_HOME set correctly), it should be possible to simply type : ant target where "target" is one of the names produced by the listing.
