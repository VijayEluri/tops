Essential Software

You will need DocBook from OASIS and apache foundation's ant (see : Appendix A).

How to Build

The ant target "manual" is the one to use. It will first delete the old copy of the manual (move it somewhere else if you want to be safe!), and then attempt to parse the XML files in docs/manual-src and generate the HTML in docs/manual.
